# ArıKitap

ArıKitap, kullanıcıların kitapları görüntüleyebildiği, yorum yapabildiği ve sepetine ekleyebildiği PHP tabanlı bir kitap satış web sitesidir. Arı temalı görsel tasarımıyla kullanıcı dostu bir alışveriş deneyimi sunar.

## Özellikler

- **Kitap Listesi:** Anasayfada kitapların listelenmesi ve detaylarının gösterilmesi.
- **Kullanıcı Girişi:** Kayıtlı kullanıcıların giriş yapabilmesi.
- **Yorum ve Puanlama:** Kullanıcılar kitaplara yorum yapabilir ve puan verebilir.
- **Sepet Sistemi:** Kitaplar sepete eklenebilir ve sepetteki ürünler görüntülenebilir.
- **Favoriler:** Kitaplar favorilere eklenebilir.
- **Arı Teması:** Site tasarımı arı konsepti ile görsel olarak desteklenmiştir.

## Teknolojiler

- **Backend:** PHP  
- **Veritabanı:** MySQL (`kitap_satis`)  
- **Frontend:** HTML, CSS, JavaScript
